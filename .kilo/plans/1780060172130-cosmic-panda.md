# Plan: Fix Layout & Styling Issues in EmissioTrace Landing Page

## Objective

Fix the "layout broken" issue by correcting CSS syntax errors and verifying Tailwind/Nuxt UI processing.

## Issues Identified (from Code Inspection)

### 1. CSS Property Typos in `app/assets/css/main.css`

**Lines 66-68** (from `cat -n` output):

```css
html {
  scroll-behavior: smooth; /* WRONG - missing 'i' in behavior */
  -webkit-font-smoothing: antialiased; /* WRONG - missing 'n' in smoothing */
}
```

**Correct spelling**:

```css
html {
  scroll-behavior: smooth;
  -webkit-font-smoothing: antialiased;
}
```

These typos will cause these CSS properties to NOT be applied, breaking smooth scrolling and font rendering.

### 2. Custom Fonts Not Loading

**Lines 7-8**:

```css
--font-display: "Roobert PRO Medium", "Roobert PRO", -apple-system, BlinkMacSystemFont, sans-serif;
--font-body: "Noto Sans", -apple-system, BlinkMacSystemFont, "Segoe UI", system-ui, sans-serif;
```

"Roobert PRO" and "Noto Sans" are not imported/installed, so browsers will fallback to system fonts. This affects `.eyebrow`, `h1-h4`, `.font-display` elements.

### 3. Potentially Broken `@theme` Block Processing

The `@theme` block (lines 4-34) defines custom properties for Tailwind v4, but if CSS has syntax errors (like the typos above), the entire file may not be processed correctly, causing `bg-bg`, `text-muted`, `border-ring` classes to NOT be generated.

## Implementation Steps

### Step 1: Fix CSS Typos in `app/assets/css/main.css`

**CRITICAL**: Fix lines 66-68:

```css
  -webkit-font-smoothing: antialiased;

```

Use the `edit` tool to replace the incorrect properties with correct spelling.

### Step 2: Fix Font Configuration

**Option A (Recommended)**: Remove custom fonts, use system fonts only:

Edit `app/assets/css/main.css` `@theme` block (lines 4-8):

```css
@theme {
  /* Fonts - system fonts only */
  --font-sans: -apple-system, BlinkMacSystemFont, "Segoe UI", system-ui, sans-serif;
  --font-display: -apple-system, BlinkMacSystemFont, "Segoe UI", system-ui, sans-serif;
  --font-body: -apple-system, BlinkMacSystemFont, "Segoe UI", system-ui, sans-serif;

  /* Rest of theme variables... */
}
```

**Option B**: Add Google Fonts import for "Noto Sans":

```css
@import url("https://fonts.googleapis.com/css2?family=Noto+Sans:wght@400;600;700&display=swap");
```

### Step 3: Remove Unused `:root` Block

The `:root` block (lines 37-55) defines `--space-*` variables but they're NOT being used in the components (components use Tailwind classes like `mb-6`, `gap-3`, etc.). Remove this block to simplify the CSS.

### Step 4: Verify Tailwind Classes are Generated
